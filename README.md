# AI CCTV Accident Detection using YOLO and Grid Division Method

## Project Overview
This project detects road accidents from CCTV footage using YOLO object detection and a grid division method. It tracks vehicle movement in real time and identifies possible accident events.

## Features
- Real-time vehicle detection
- Accident detection using movement analysis
- Grid-based frame monitoring
- Automated email alert to nearby patrol teams

## Technologies Used
- Python
- YOLO
- OpenCV
- Jupyter Notebook

## How It Works
1. CCTV video is processed frame by frame.
2. YOLO detects vehicles in each frame.
3. Grid division is used to analyze vehicle position and movement.
4. If an accident is detected, the system sends an email alert to nearby patrol teams.
